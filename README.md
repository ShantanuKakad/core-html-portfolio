# Personal Portfolio Website

This repository contains my personal portfolio website built using **HTML5** and **CSS3**.  
It showcases my academic background, technical skills, learning journey, and professional interests in a clean multi-page format.

## Project Overview

This portfolio website is designed to present who I am as a **Computer Engineering student** and an aspiring technology professional.  
It highlights my education, current skill set, academic progress, and areas of interest such as:

- Cloud Computing
- Cyber Security
- DevOps
- Software Engineering

The project reflects my learning-by-building approach and documents my growth in programming, Linux, Git, and web development.

## Features

- Home page with personal introduction and focus areas
- Education page covering academic and self-learning journey
- Skills page showing current technical strengths
- Academic Snapshot page showing semester performance
- Contact page with email and message form
- GitHub and LinkedIn profile links
- Clean multi-page website structure
- Built using core HTML and custom CSS

## Technologies Used

- HTML5
- CSS3

## Project Structure

- `index.html` – Main homepage
- `education.html` – Education and learning journey
- `skills.html` – Skills and current stack
- `marks.html` – Academic snapshot
- `contact.html` – Contact details and message form
- `Photograph.jpeg` – Profile image
